# Simple Static Website

This project is a simple static website that includes HTML, CSS, and JavaScript files. It serves as a basic template for creating a web application.

## Project Structure

```
simple-static-website
├── src
│   ├── index.html         # Main HTML document
│   ├── css
│   │   └── styles.css     # Styles for the website
│   ├── js
│   │   └── main.js        # JavaScript code for interactivity
│   └── assets
│       ├── fonts          # Directory for font files
│       └── data           # Directory for data files (e.g., JSON, images)
├── .gitignore             # Git ignore file
└── README.md              # Project documentation
```

## Getting Started

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd simple-static-website
   ```

3. Open the `src/index.html` file in your web browser to view the website.
